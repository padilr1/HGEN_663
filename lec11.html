Call peaks, run IDR and then pool samples

Call peaks using MACS3

macs3 callpeak -g hs -f BAMPE --extsize 200 --shift -100 --nomodel -B -p 1e-3 --keep-dup all -t mut_rep1.bam -n mut_rep1 --outdir peaks

Run IDR

idr --samples mut_rep1.narrowPeak mut_rep2.narrowPeak \
--input-file-type narrowPeak \
--rank p.value \
--output-file mut-idr \
--output-file-type bed \
--idr-threshold 0.05 \
--plot \
--log-output-file mut.idr.log

Pool together peaks. Make a .saf file to use with featureCounts

cat *-idr | sort -k1,1 -k2,2n | bedtools merge | tee merged.bed | wc -l
bedtools intersect -a merged.bed -b blacklist.hg19.bed -v \
> merged.filtered.bed
awk 'OFS="\t" {print $1"."$2"."$3, $1, $2, $3, "."}' merged.filtered.bed > merged.saf

Run featureCounts

featureCounts -a merged.saf -F SAF -o xie2022.fc.txt -T 6 -p mut_rep1.bam mut_rep2.bam rescue_rep1.bam rescue_rep2.bam
Assess profile of ATAC-seq signal on open and closed peaks

Aggregate signal centered on open and closed ATAC-seq peaks using computeMatrix

computeMatrix reference-point -R open_peaks.bed \
-S mut_rep1.cpm.bw mut_rep2.cpm.bw rescue_rep1.cpm.bw rescue_rep2.cpm.bw \
-bl blaclist.hg19.bed \
--referencePoint center \
--binSize 50 -a 2500 -b 2500 \
--missingDataAsZero --skipZeros \
--samplesLabel mut_1 mut_2 rescue_1 rescue_2 \
-o ATAC.mat.gz

Visualize output through plotHeatmap

plotHeatmap -m ATAC.mat.gz -o heatmap.png --dpi 600 --colorMap Reds --verbose -x "ATAC" -y "" --startLabel "" --endLabel "" --regionsLabel ""
Run HOMER on the open and closed ATAC-seq peaks

Perform motif enrichment analysis using findMotifsGenome.pl

findMotifsGenome.pl open_peaks.bed hg19 HOMER.open_peaks -size given -mask -bg merged.filtered.bed
